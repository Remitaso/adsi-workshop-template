# 画面設計

## 画面一覧

| # | 画面名 | パス | ロール | 説明 |
|---|--------|------|--------|------|
| S-01 | ログイン | `/login` | 全員 | ID/パスワード入力 |
| S-02 | ダッシュボード | `/` | 全員 | 打刻ボタン + 本日の勤怠状態 |
| S-03 | 月次勤怠一覧 | `/attendance` | 全員 | カレンダー形式で日別勤怠 |
| S-04 | 打刻修正 | `/attendance/{date}` | 全員 | 打刻エントリの修正 |
| S-05 | 休暇申請 | `/leaves/new` | 全員 | 休暇種類・期間を入力して申請 |
| S-06 | 休暇一覧 | `/leaves` | 全員 | 自分の休暇申請と残高 |
| S-07 | 承認一覧 | `/approval` | 管理者 | 打刻・休暇の承認待ち一覧 |
| S-08 | チームダッシュボード | `/team` | 管理者 | チーム全員の勤怠状況 |
| S-09 | 月次レポート | `/reports` | 人事 | 全社集計 + CSV エクスポート |
| S-10 | 社員管理 | `/admin/employees` | 人事 | 社員 CRUD |
| S-11 | 組織管理 | `/admin/organizations` | 人事 | 部・課・チーム管理 |

## 共通コンポーネント

| コンポーネント | 説明 |
|--------------|------|
| Header | ナビゲーション + 通知バッジ + ユーザーメニュー |
| Sidebar | 管理者・人事はサイドメニュー表示 |
| AlertBanner | 残業アラート・打刻忘れ通知 |
| StatusBadge | DRAFT/SUBMITTED/APPROVED の状態表示 |
| ConfirmDialog | 承認・却下等の確認ダイアログ |
| DataTable | ソート・ページネーション対応テーブル |
| MonthPicker | 年月選択 |

  組織管理 (/admin/organizations)
