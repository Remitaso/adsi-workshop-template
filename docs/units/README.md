                    ┌─────────────────┐
                    │  Unit 0: 共通基盤 │

   ┌────────────────────┐   ┌────────────────────┐
   │ Unit 1: 打刻・勤怠  │   │ Unit 2: 休暇管理   │

## 実装順序

| Phase | Unit | 担当 | 依存 | 並行可否 |
|-------|------|------|------|---------|
| 1 | Unit 0: 共通基盤 | 2人協力 | なし | — |
| 2 | Unit 1: 打刻・勤怠 | 担当者 A | Unit 0 | ✓ 並行 |
| 2 | Unit 2: 休暇管理 | 担当者 B | Unit 0 | ✓ 並行 |
| 3 | Unit 3: 集計・レポート | 2人分担 | Unit 1 + 2 | — |

## 各 Unit の概要

### [Unit 0: 共通基盤](unit_0_foundation.md)
- プロジェクトスケルトン（Gradle + Next.js）
- DB接続 + Flyway + 組織マスタテーブル
- Spring Security（認証API）
- 共通エラーハンドリング
- テスト基盤（ArchUnit含む）

### [Unit 1: 打刻・勤怠記録](unit_1_attendance.md)
- 出退勤打刻（中抜け対応）
- 打刻修正 + 承認フロー
- 勤務時間自動計算（残業・深夜割増）
- ダッシュボード + 月次勤怠一覧画面

### [Unit 2: 休暇管理](unit_2_leave.md)
- 休暇申請（5種類）+ 承認/却下
- 有給残高管理（勤続年数で自動付与）
- 休暇申請画面 + 休暇一覧画面

### [Unit 3: 集計・レポート・アラート](unit_3_reporting.md)
- 月次勤怠集計
- CSV エクスポート
- 残業アラート（月45時間超）
- 打刻忘れ通知
- チームダッシュボード + 月次レポート画面

## 見積もり目安

| Unit | 規模 | Backend | Frontend |
|------|------|---------|----------|
| Unit 0 | M | テーブル4 + 認証 + CRUD | ログイン + レイアウト |
| Unit 1 | L | テーブル2 + 時間計算ロジック | 画面3つ |
| Unit 2 | M | テーブル2 + 付与ロジック | 画面2つ |
| Unit 3 | L | 集計 + CSV + スケジューラ | 画面2つ + アラート |
